Un index simple pointant vers les images ISO officielles et les pages de vérification (checksums / signatures).

Important : Télécharge toujours depuis la page officielle et vérifie les checksums/signatures avant usage.

Comment vérifier une ISO (rapide)

Procédure générale (Linux/macOS/WSL) — pour Windows voir la ligne PowerShell plus bas.

  1. Téléchargez l’ISO + le fichier de sommes (ex : SHA256SUMS ou SHA256SUMS.gpg) depuis le site officiel.
  2. Vérifiez la signature GPG (si fournie) : gpg --verify SHA256SUMS.gpg SHA256SUMS
  3. Vérifiez le hash : sha256sum -c SHA256SUMS # ou sha512sum -c SHA512SUMS
  4. Sur macOS : shasum -a 256
  5. Sur Windows PowerShell : Get-FileHash -Path .\nom.iso -Algorithm SHA256

Si vous avez seulement une valeur de hash affichée (par ex. sur la page web), comparez-la au résultat de la commande (exact match).

Si la page officielle fournit une signature GPG pour le fichier de sommes, vérifiez-la toujours (empêche la falsification des sommes).
